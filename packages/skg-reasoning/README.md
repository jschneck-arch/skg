# skg-reasoning

Canonical SKG reasoning layer (`SKG-R`).

Scope:
- consume canonical domain events and protocol-governed context contracts
- derive higher-order reasoning outputs (path pressure, value pressure, attacker usefulness)
- keep reasoning explicit, versioned, and testable

Non-scope:
- redefining domain semantics
- emitting raw observations/domain events
- runtime collection/transport/orchestration
- legacy adapter coupling
