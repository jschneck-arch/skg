- the fallback refits on every new text

- direct validation confirmed a text's vector changes after corpus growth, and a drifting-embedder `MemoryStore` test confirmed ranking can flip under that assumption

Conclusion:

The fallback resonance path is currently only reliable if a stable embedder is available or the index is rebuilt aggressively.

These are live correctness risks with little regression protection.
