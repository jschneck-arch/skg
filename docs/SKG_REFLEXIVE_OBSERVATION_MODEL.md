# SKG Reflexive Observation Model

Purpose: state clearly that SKG is part of the measurement apparatus and therefore subject to observation, insufficiency, and growth.

## Principle

SKG is not outside its own observation.

The target is observed through SKG’s instruments, mappings, projections, memory, and operator surfaces.
Therefore inadequacies in SKG are themselves part of the observed system.

## What This Means

A fold can indicate:

- insufficiency in the target model
- insufficiency in an instrument
- insufficiency in the catalog
- insufficiency in projection structure
- insufficiency in recall/memory
- insufficiency in operator-facing aggregation

These are not outside the substrate.
They are legitimate substrate conditions.

## Examples

Examples of reflexive insufficiency include:

- a service is present but no toolchain exists
- an observed CVE has no wicket mapping
- an attack chain is implied but no projection exists
- an instrument fails to preserve a realized condition because of truncation or bad ordering
- an operator surface fragments one structural deficit into many noisy work items
- state is written in one format but read in another

Each of these is a valid object of SKG observation.

## Why This Matters

Without this reflexive frame, SKG could audit targets while silently exempting its own apparatus from the same discipline.

That would make it a narrative system rather than an observational substrate.

The reflexive model prevents that.

## Operational Sequence

The correct sequence is:

1. observe insufficiency
2. represent insufficiency
3. preserve insufficiency
4. expose insufficiency to the operator
5. route attention toward it
6. later, under bounded conditions, assist in correcting it

Immediate autonomous self-correction is not the first step.

## Role In The Proving Ground

A real SKG proving ground must include:

- target-side insufficiency
- instrument-side insufficiency
- model-side insufficiency
- cases where SKG must remain indeterminate
- cases where SKG must improve its own structure to continue measuring well

That is how SKG proves it is measuring rather than narrating.
